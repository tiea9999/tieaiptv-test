# tieaiptv-test

Proxy สำหรับดูช่อง TV ออนไลน์จาก dookeela / neoistheone
พร้อมดึง token / m3u8 อัตโนมัติ

## วิธีใช้

1. Upload ไฟล์นี้ขึ้น GitHub
2. Deploy บน Render / Railway / Vercel
   - Environment: Node
   - Start Command: node server.js
   - Region: Singapore
3. หลัง deploy เสร็จ URL จะเป็นเช่น:
   https://<your-app>.onrender.com/proxy?channel=warnertv
